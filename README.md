

## To begin:

### 1. Run Composer Install
### 2. Duplicate .env.example
### 3. Rename .env.example copy to .env
### 4. Create a database
### 5. Set DB_DATABASE in .env to database name
### 6. Run php artisan run:setup

#### That's it. You are good to go

## To run tests
### Run php artisan test
